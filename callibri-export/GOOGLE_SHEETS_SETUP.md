# Подключение Google Sheets к Callibri Export — пошаговая инструкция

## Шаг 1. Создать проект в Google Cloud

1. Открой [console.cloud.google.com](https://console.cloud.google.com/)
2. Войди под своим Google-аккаунтом
3. Вверху нажми на выпадающий список проектов (рядом с "Google Cloud") → **«Новый проект»**
4. Название — любое, например `Callibri Export` → **«Создать»**
5. Убедись, что новый проект выбран в верхнем выпадающем списке

## Шаг 2. Включить Google Sheets API

1. В левом меню → **«API и сервисы»** → **«Библиотека»**
2. В поиске напиши `Google Sheets API`
3. Кликни на него → **«Включить»** (синяя кнопка)
4. Подожди пару секунд — появится надпись «API включён»

## Шаг 3. Создать сервисный аккаунт

1. В левом меню → **«API и сервисы»** → **«Учётные данные»**
2. Вверху нажми **«+ Создать учётные данные»** → **«Сервисный аккаунт»**
3. Заполни:
   - Название — любое, например `callibri-export`
   - ID сервисного аккаунта — заполнится автоматически
4. Нажми **«Создать и продолжить»**
5. Остальные шаги (роли, доступ) — **пропусти**, просто жми **«Готово»**

## Шаг 4. Скачать ключ (credentials.json)

1. На странице **«Учётные данные»** — в разделе «Сервисные аккаунты» появился твой аккаунт
2. Кликни на его email (вида `callibri-export@...iam.gserviceaccount.com`)
3. Вверху перейди на вкладку **«Ключи»**
4. Нажми **«Добавить ключ»** → **«Создать новый ключ»**
5. Тип — **JSON** → **«Создать»**
6. Файл скачается автоматически (называется что-то вроде `project-name-abc123.json`)
7. **Переименуй** его в `credentials.json`
8. **Положи** его в папку рядом с `CallibriExport.exe`

## Шаг 5. Дать доступ к таблице

1. Открой файл `credentials.json` блокнотом — найди строку `"client_email":`
   - Там будет что-то вроде `callibri-export@my-project.iam.gserviceaccount.com`
   - **Скопируй** этот email
2. Открой нужную **Google-таблицу** в браузере
3. Нажми **«Настройки доступа»** (зелёная кнопка справа вверху)
4. Вставь скопированный email → роль **«Редактор»** → **«Отправить»**

## Шаг 6. Настроить в приложении

1. Запусти `CallibriExport.exe`
2. В секции **«Google Sheets»** → поле «Credentials» → нажми **`...`** → выбери `credentials.json`
3. Нажми **«Проверить»** — должно появиться «Подключено. Service Account: ...»
4. У нужного проекта нажми **«Настроить»** → вкладка **«Google Sheets»**
5. Включи галку **«Отправлять в Google Sheets»**
6. Вставь **ссылку** на Google-таблицу (или её ID из URL)
7. Нажми **«Загрузить листы»** — появится список вкладок
8. Выбери нужный лист
9. Выбери режим: **append** (дополнять данные) или **replace** (заменять)
10. Нажми **«Сохранить»**

Теперь при каждом экспорте данные будут отправляться и в файл, и в Google-таблицу.
